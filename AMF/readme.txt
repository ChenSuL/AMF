This directory maintains the core implementation of adaptive matrix 
factorization (AMF) using C++, as well as the wrapper using Cython.

****************************************************************************
Contents of this directory
****************************************************************************

AMF/
  - readme.txt     - descriptions of this directory 
  - c_AMF.h        - header file
  - c_AMF.cpp      - source file
  - AMF.pyx        - a python wrapper written in Cython
  - AMF.cpp        - file automatically generated by Cython
  - __init__.py    - a file to initialize Python package 

****************************************************************************
Reference and citation
****************************************************************************

Please refer to the following paper for the detailed description of the 
implemented algorithm:

- Jieming Zhu, Pinjia He, Zibin Zheng, and Michael R. Lyu, "Towards Online, 
  Accurate, and Scalable QoS Prediction for Runtime Service Adaptation," in 
  Proc. of IEEE ICDCS, 2014, pp. 318-327.
